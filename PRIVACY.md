# Privacy Policy for Excel Process Plugin

Last Updated: June 23, 2025

## What This Plugin Does

This plugin processes Excel (`.xlsx`/`.xls`) files you provide to extract
textual cell content and embedded images.

## Data Processed

- Excel files you upload
- Text and metadata from those spreadsheets
- Images embedded inside the workbook

## Not Collected

No personal information is collected or stored, including:
- Names or contact details
- Login credentials
- Device information
- Location data

## Data Handling

- Files are processed in memory only
- No data is stored after processing completes
- Uses openpyxl, xlrd, and Python's zipfile/olefile modules for Excel parsing
- Extracted images are temporary and deleted after processing
- Your data is not shared with anyone

## Your Control

You control what Excel files you provide and can stop using the plugin at any time.
